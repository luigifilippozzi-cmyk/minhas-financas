# 🔗 Projetos Open Source de Referência

## Projetos Usados como Base

### Firebase Extended — Expense Tracker
- **Repositório:** https://github.com/FirebaseExtended/expense-tracker
- **Por que usar:** Projeto oficial do Firebase, código limpo, JavaScript vanilla
- **O que adaptar:**
  - Estrutura de autenticação Firebase
  - Padrão de interação com Firestore
  - Organização de arquivos JS
  - Configuração Firebase
- **O que NÃO copiar:**
  - Categorias fixas (nosso sistema é dinâmico)
  - Sistema de grupos (não existe no projeto base)
  - Orçamentos (não existe no projeto base)

### Finapp
- **Repositório:** https://github.com/ilkome/finapp
- **Referência para:** UI/UX de dashboards financeiros

### FamilyFinance
- **Repositório:** https://github.com/farminf/FamilyFinance
- **Referência para:** Gestão financeira multi-usuário/familiar
